# My Portfolio

A modern and responsive portfolio website built with React, SCSS, and Framer Motion.

## 🚀 Live Demo

Portfolio Website: https://faeze-danesh.vercel.app/

## ✨ Features

* Responsive design for desktop, tablet, and mobile
* Modern UI with smooth animations
* Projects showcase section
* Skills and About Me sections
* Contact section
* Built with reusable React components

## 🛠 Technologies Used

* React.js
* SCSS
* Framer Motion
* Vite
* Git & GitHub
